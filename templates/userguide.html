Welcome to the User Guide

This guide will help you navigate and use the PAANN T2D SNP Browser efficiently.

Searching for SNPs

You can search for Type 2 Diabetes-related SNPs using:

  • RSID (e.g., rs148776975)
  • Gene Name (e.g., RSPO3)
  • Gneomic Region (e.g., Chr :15, Start: 62000000, End: 63000000)

Use the search bar on the home page to enter your query and get relevant results.

Filtering Results

After performing a search, you can filter results based on:

  • Phenotypes (e.g., Fasting Glucose, HbA1C)
  • Populations (e.g., Bengali, Punjabi)

Interpreting Results

The results page provides detailed information, including:

  • SNP Details: ID, associated gene, chromosome position
  • Risk Allele Frequency: Population-specific risk data
  • Statistical Significance: P-values
  • Visualisations: Visualisations available for genomic regions of interest
