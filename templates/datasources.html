Data Sources

Explore the data sources used to identify Type 2 Diabetes (T2D) genetic variants.

Our Key Data Sources

  • SNP Data from 1000 Genomes: We downloaded VCF files containing bi-allelic SNP data for Punjabis from Lahore (PJL) and Bengalis (BEB) from the 1000 Genomes Project.
  • T2D KP: Using the Type 2 Diabetes Knowledge Portal, we obtained a list of SNPs linked to T2D-related phenotypes and filtered the VCF files accordingly.
  • Scientific Literature: Peer-reviewed studies provided supporting evidence for genetic associations, helping us interpret GWAS findings in the context of T2D [1], [2], [3].
  • PAN-GO Database:We used PAN-GO for our gene ontology database, which categorises genes based on molecular function, biological process, and cellular component.
  • Ensembl: When PAN-GO did not have gene ontology terms for certain genes, we used Ensembl to obtain alternative gene data, ensuring comprehensive annotation of genetic variants.
